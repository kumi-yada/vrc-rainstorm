# Join Sound

- Author: Purabe Works
- Date: 2026/02/05

VRChat ワールド用 UdonSharp ギミックです。  
自分以外のプレイヤーがワールドにjoinした際に、指定したサウンドを再生します。

## 使い方

- JoinSound.prefab をシーンに配置します。
- 音源を変更する場合、 `JoinSoundAudioClip` を差し替えます。
